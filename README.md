# 🌿 Koral-Mat — משחק חיבורים לצמחי מרפא סיניים

משחק לימוד בהשראת [NYT Connections](https://www.nytimes.com/games/connections) לשינון המטריה מדיקה — מיון צמחים לקטגוריות שלהם.

**🎮 לשחק עכשיו:** [koralmt-tcm.github.io/Koral-Mat](https://koralmt-tcm.github.io/Koral-Mat/)

עובד מצוין גם בנייד — פשוט לפתוח את הלינק בדפדפן.

## איך משחקים

1. **בוחרים בלוק סילבוס** (מבחן ראשון / שני / שלישי) — המשחק מגריל 4 קטגוריות מתוך הבלוק
2. **בוחרים רמת קושי:**
   - 🟢 **קל** — רק צמחי A
   - 🟡 **בינוני** — צמחי A+B
   - 🔴 **קשוח** — כל הצמחים (A+B+C)
3. על הלוח 16 צמחים מ-4 קטגוריות — מסמנים 4 שלדעתכם שייכים לאותה קטגוריה ולוחצים **שלח**
4. יש **6 לבבות** לכל סיבוב — טעות מורידה לב. "קרוב מאוד..." אומר ש-3 מתוך 4 נכונים
5. המשחק ממשיך סיבוב אחרי סיבוב עד שכל הצמחים בקטגוריות שנבחרו נלמדו 🏆

## 💡 פיצ'ר הלמידה

**לחיצה ארוכה על צמח** (או קליק ימני במחשב) פותחת את **התמצית** שלו — תיאור קליני קצר בעברית: טבע, טעם, פעולות עיקריות ואזהרות. ככה אפשר לרענן את הזיכרון בלי לצאת מהמשחק.

## מה יש בפנים

- 237 צמחים ב-24 קטגוריות, מחולקים ל-3 בלוקי סילבוס
- כל התוכן בקובץ HTML אחד — בלי שרת, בלי התקנה, עובד גם אופליין אחרי טעינה ראשונה
- צמחים שכבר הופיעו יכולים לחזור בסיבובים מתקדמים כהשלמה — בלי סימון מיוחד, אז תמיד צריך לזהות אותם מחדש 😉
